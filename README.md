# 2048 - SASCO Games Edition

A fun number puzzle game for students! Part of the SASCO Games collection for schools.

## How to Play

Use your **arrow keys** to move the tiles. When two tiles with the same number touch, they **merge into one!** The goal is to reach the **2048 tile**.

## Controls

- **Arrow Keys** - Move tiles up, down, left, or right
- **New Game Button** - Start a fresh game
- **Keep Playing** - Continue after reaching 2048

## Features

- Smooth animations
- Score tracking with best score saved
- Mobile-friendly (touch/swipe support)
- Works offline
- Clean, colorful design for kids

## Installation

Simply open `index.html` in any modern web browser. No build process or dependencies needed!

## For Schools

This game helps develop:
- Strategic thinking
- Planning ahead
- Pattern recognition
- Basic math skills (doubling numbers)
- Problem-solving
